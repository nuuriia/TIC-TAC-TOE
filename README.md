# TIC-TAC-TOE
First-year project for the course Programació i Algorísmia 1 in the Bachelor’s Degree in Artificial Intelligence (FIB, UPC). Advanced Tic-Tac-Toe implemented in Python with configurable board size, number of stones, Classic and Misery modes, terminal and Pygame GUI versions, plus a simple AI opponent.

The game allows configurable board size and number of stones per player, includes Classic and Misery modes, and is divided into two phases: stone placement and stone movement. The logic is adaptable to different board dimensions.

The project includes:
- A terminal-based version
- A graphical version implemented with Pygame
- An optional single-player mode against a simple AI with basic strategic behavior

How to run
Terminal version:
  **python main.py**
  
Graphical version:
  **python main_gui.py**

Make sure you have Python 3 installed. For the graphical version, install Pygame:
  pip install pygame
